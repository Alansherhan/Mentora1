## ðŸ”§ Troubleshooting

### Common Issues:

#### Chatbot Not Responding:
- **Check Internet Connection**: Ensure you're connected to the server
- **Refresh Page**: Try refreshing your browser
- **Check Server Status**: Ensure the server is running (check with admin)

#### File Upload Issues:
- **Check File Format**: Only PDF files are supported
- **Check File Size**: Files should be under 10MB
- **Check Permissions**: Ensure the uploads folder has write permissions

#### Search Not Working:
- **Check Keywords**: Ensure materials have relevant keywords
- **Try Different Terms**: Use synonyms or related terms
- **Check Spelling**: SmartBuddy handles typos, but exact matches work better

#### Admin Access Issues:
- **Check Password**: Verify you're using the correct password
- **Clear Browser Cache**: Try clearing your browser cache
- **Check Server**: Ensure the server is running properly
